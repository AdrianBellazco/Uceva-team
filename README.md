## Cómo correr el proyecto

### 1. Instalar dependencias
```bash
npm install
```

### 2. Levantar el servidor de desarrollo
```bash
ng serve
```

Luego abre http://localhost:4200 en tu navegador.

### 3. Build de producción
```bash
ng build
```
